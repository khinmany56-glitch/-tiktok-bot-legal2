Privacy Policy

Last updated: April 2026

1. Introduction

This Privacy Policy explains how this personal content automation application handles your information when you authorize it to access your TikTok account.

2. Information We Collect

We collect only the minimum information required to operate the application:

  • TikTok OAuth Token — stored locally on your device to authenticate API requests
  • TikTok Open ID — a non-identifiable user identifier provided by TikTok

We do not collect your name, email address, phone number, password, or any personal profile information.

3. How We Use Your Information

The OAuth token is used solely to:

  • Upload video files to your TikTok account via the Content Posting API
  • Check the publish status of uploaded videos

4. Data Storage

All data is stored locally on your own device in the file tiktok_token.json. No data is transmitted to any external server other than TikTok's official API endpoints.

5. Data Sharing

We do not sell, trade, or share your data with any third parties. Your TikTok credentials are never shared outside of TikTok's official API.

6. TikTok API Permissions

This application requests the following TikTok API scopes:

  • video.upload — to upload video files
  • video.publish — to publish videos to your account

You can revoke these permissions at any time from your TikTok account settings under Security > Manage app permissions.

7. Data Retention

The OAuth token is retained locally until you explicitly log out or delete the tiktok_token.json file. Tokens automatically expire as defined by TikTok's API policy.

8. Your Rights

You have the right to:

  • Revoke API access at any time via TikTok's app settings
  • Delete your locally stored token by removing tiktok_token.json
  • Request information about what data is stored

9. Changes to This Policy

This policy may be updated occasionally. The "Last updated" date at the top of this page will reflect any changes.

10. Contact

If you have questions about this Privacy Policy, please contact the application owner directly.
